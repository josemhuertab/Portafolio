# Imágenes de Proyectos

Coloca aquí las imágenes de tus proyectos.

## Recomendaciones:

- **Formato**: JPG, PNG o WebP
- **Dimensiones recomendadas**: 1920x1080px (16:9)
- **Tamaño máximo**: 500KB por imagen (optimiza tus imágenes)
- **Nombres**: Usa nombres descriptivos sin espacios (ej: `ecommerce-app.jpg`)

## Herramientas para optimizar imágenes:

- [TinyPNG](https://tinypng.com/) - Compresión de PNG y JPG
- [Squoosh](https://squoosh.app/) - Optimizador de imágenes de Google
- [ImageOptim](https://imageoptim.com/) - Para macOS

## Ejemplo de uso en projects.tsx:

```typescript
{
  title: 'Mi Proyecto',
  image: '/projects/mi-proyecto.jpg', // La ruta comienza con /projects/
  // ...
}
```
